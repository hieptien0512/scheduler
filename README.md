# Install XAMPP
Install XAMPP from https://www.apachefriends.org/download.html
Start Apache server
Start MySQL server

# Prepare Database with PHPMyAdmin
Run script from file scheduler.sql at http://localhost/phpmyadmin/index.php?route=/server/sql
